# MinQ 100万ダウンロード達成のための機能戦略案 (Growth Strategy)

## 1. 戦略概要: 「個人の習慣」を「社会的なムーブメント」へ

現在のMinQは「個人の習慣化」と「1対1のペア」に焦点を当てていますが、100万ダウンロードという爆発的な成長（バイラル）を生むためには、**「シェアしたくなる動機」**と**「辞められない仕組み（社会的圧力）」**を強化する必要があります。

**3つの柱:**
1.  **Acquisition (獲得):** 招待したくなる「メリット」と「コンテンツ」
2.  **Retention (継続):** 1人では味わえない「共鳴（Resonance）」体験
3.  **Engagement (没入):** AIによるパーソナライズされた「世界観」

---

## 2. 提案機能 (Growth Engines)

### 🚀 Feature 1: 「絆パス (Kizuna Pass)」招待システム
**目的: バイラル係数（K-factor）の向上**

*   **課題:** 現在のペアマッチングはランダム性が高く、リアルの友人を誘う導線が弱い。
*   **提案:**
    *   ユーザーに**「絆パス（招待コード）」**を付与する。
    *   このパスを使って友人を招待すると、2人は即座にペアとしてマッチングされ、**「1週間のプレミアム機能（広告非表示や限定テーマ）」**が開放される。
    *   **心理効果:** 「アプリを勧める」のではなく「プレミアムチケットをあげる」というギフトの文脈に変えることで、招待の心理的ハードルを下げる。

### 🎨 Feature 2: AIハビットDNA診断 (Shareable Content)
**目的: SNSでの認知拡大 (Buzz)**

*   **課題:** 「今日もクエスト完了しました」というシェアは退屈で、承認欲求を満たしにくい。
*   **提案:**
    *   ユーザーの行動データ（活動時間、選ぶクエストの種類、継続率）をAIが分析し、**「あなたの習慣タイプ」**を生成する。
    *   **例:** 「あなたは『深夜の哲学者』タイプ（夜型 x 読書習慣）。相性が良いのは『早起きの戦士』タイプです！」
    *   MBTI診断のように、美しくデザインされた結果画像をワンタップでInstagram/Xにシェアできる機能を実装する。
    *   **実装:** `OgpImageGenerator` を拡張し、AI生成のキャッチコピーを載せる。

### 🔥 Feature 3: 共鳴ストリーク (Resonance Streak)
**目的: 離脱率の低下 (Retention)**

*   **課題:** 個人のストリーク（継続記録）は、忙しいと簡単に途切れてしまい、一度途切れるとモチベーションがゼロになる（どうにでもなれ効果）。
*   **提案:**
    *   ペアとの**「合算ストリーク」**を導入する。2人とも達成した日だけ伸びる特別なストリーク。
    *   一方がサボりそうな時、もう一方が「助ける（クエスト代行や応援）」ことでストリークを守れる**「ライフライン機能」**を追加。
    *   **心理効果:** 「自分のためならサボるが、相手のストリークを壊すわけにはいかない」という社会的圧力をポジティブに利用する。

### 🌍 Feature 4: ワールドボス・イベント (Community)
**目的: アプリ全体の一体感**

*   **課題:** ペア以外のユーザーとの繋がりがなく、アプリの世界が閉じて見える。
*   **提案:**
    *   **「全ユーザー参加型クエスト」**を定期開催する。
    *   例: 「みんなで合計100万回スクワットして、魔王（運動不足）を倒せ！」
    *   ホーム画面にリアルタイムの討伐カウンターを表示。
    *   貢献度に応じて限定バッジやアプリアイコンを付与。
    *   **効果:** 初心者でも「参加している感」を味わえ、アプリを開く頻度が増える。

## 3. 実装ロードマップ (Roadmap)

100万DLを目指すための段階的な実装計画です。

### Phase 1: 基盤強化 (The Foundation) - *Current Priority*
*   UI/UXの刷新 (デザインシステムの適用)
*   習慣化コアループの実装 (Gamification Engineの稼働)
*   アプリの安定化 (Crash Free 99.9%を目指す)

### Phase 2: バイラルの仕込み (The Spark)
*   「AIハビットDNA診断」の実装とシェア導線の強化
*   「絆パス」招待機能の実装

### Phase 3: コミュニティの熱狂 (The Fire)
*   「共鳴ストリーク」の実装
*   「ワールドボス」イベントの開催

## 4. 結論
単なる「TODOアプリ」ではなく、**「誰かと一緒だから続けられる、AIが褒めてくれるサードプレイス」**としてのブランドを確立することが、100万ダウンロードへの最短ルートです。まずはPhase 1の「当たり前品質」をクリアし、Phase 2の「診断シェア」で初期のバズを狙うことを推奨します。
